# Körner — Cafetería & Bar

Sitio web de Körner, cafetería y bar en Punta Arenas, Chile. Incluye carrito de compras para merchandising y sistema de pedidos del menú, ambos con checkout vía WhatsApp.

## Cómo correrlo localmente

```bash
npm install
npm run dev
```

## Configuración

Antes de publicar, edita `src/app/config.ts` y reemplaza `WHATSAPP_NUMBER` por el número real del local (formato internacional, solo dígitos).

## Build de producción

```bash
npm run build
```
